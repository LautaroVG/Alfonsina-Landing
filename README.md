# Alfonsina Guibelalde - Official Landing Page

**Live Demo:** [alfonsinaguibelalde.com](https://alfonsinaguibelalde.vercel.app/)

Landing page y portfolio profesional desarrollado para la cantante y docente vocal Alfonsina Guibelalde. El proyecto está centrado en la performance, el diseño UI/UX moderno y la autogestión de contenido por parte del cliente.

## Stack Tecnológico

- **Frontend:** React + Vite
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Base de Datos (BaaS):** Supabase (PostgreSQL)
- **Íconos:** Lucide React
- **Hosting:** Vercel

## Features Principales

- **Gestión Dinámica de Shows:** Banner flotante inteligente conectado a Supabase. Consulta la base de datos en tiempo real para mostrar u ocultar fechas de shows y links a ticketeras, permitiendo que el cliente actualice la info desde el panel sin necesidad de nuevos deploys.
- **UI/UX Moderna & Responsive:** Uso de `100dvh` para un renderizado perfecto en mobile, menús sticky/absolutos dinámicos según el scroll y elementos flotantes optimizados para el alcance del pulgar en celulares.
- **Motion Design:** Transiciones suaves, *fade-ups* y carrusel infinito desarrollados con Framer Motion.
- **Formulario Integrado:** Recepción de consultas vinculada directamente mediante Formspree.
